O Surgimento da Internet

A internet surgiu na década de 1960, inicialmente como um projeto militar chamado ARPANET. Seu objetivo era criar uma rede de comunicação que funcionasse mesmo em caso de falhas.

Ao longo dos anos, a internet evoluiu rapidamente e passou a conectar pessoas no mundo todo.

Como Funciona o HTML

O HTML é uma linguagem de marcação que estrutura as páginas da web. Ele não é uma linguagem de programação, mas sim de marcação de conteúdo.

O HTML organiza o conteúdo por meio de tags, que são comandos interpretados pelos navegadores.

O que são Tags?

As tags são elementos que dizem ao navegador como exibir o conteúdo. Elas sempre estão entre sinais de menor e maior (< >).

Por exemplo, a tag <p> serve para criar um parágrafo.

Tags de Títulos:

  1. <h1> até <h6> - Títulos

Tags de Texto:

  • <p> - Parágrafo
  • <strong> - Negrito
  • <i> - Itálico
  • <u> - Sublinhado
  • <mark> - Marca texto
  • <small> - Texto pequeno
  • <del> - Texto deletado (riscado)
  • <sub> - Subscrito (abaixo)
  • <sup> - Sobrescrito (acima)
  • <blockquote> - Citação
  • <abbr> - Abreviação com explicação

Curiosidade

Antigamente, a tag font era muito usada para definir fonte, cor e tamanho de texto, mas hoje ela é considerada obsoleta e foi substituída por CSS.
